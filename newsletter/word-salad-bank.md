# Wall Street Word Salad — sourcing bank

The 10x value of this segment is a **real, verbatim, attributed** quote. That is
also the one place it can go fatally wrong, so one rule, no exceptions:

> **Never invent a quote and attach a real company to it.** Quote verbatim from a
> primary source, attribute it (company, document/call, date), and link it. A
> misquote pinned to a real firm isn't satire — it's the exact dishonesty this
> brand exists to mock, and it's legally radioactive. Quoting a *short* line to
> critique it is textbook fair use; making one up is not.

The archetypes below (the ones currently sitting in the issue drafts) are
**illustrative, unattributed paraphrases** — safe to ship as generic, but the paid
upgrade is to swap each for a sourced quote. The *Translation* and *What it's
hiding* lines are reusable as-is; only the quote needs replacing.

## Where to get the real ones (all free, all quotable)

- **[Salad Scout](../salad-scout/)** — your firehose. `edgar` mode searches SEC
  full-text and ranks the densest sentences with citations. (Runs anywhere with
  outbound network; SEC blocks this sandbox's proxy, so run it locally / on the NAS.)
- **Earnings-call transcripts** — IR pages and reputable transcribers. Execs dodging
  analysts is salad on tap.
- **Fed / FOMC statements & speeches** — public domain, peak hedge-speak. The latest
  statement (e.g. [June 17, 2026](https://www.federalreserve.gov/newsevents/pressreleases/monetary20260617a.htm))
  is a perfect, citation-safe source for the rates issue.

## The 10-minute workflow per issue

1. Run Salad Scout (or open the source above) and grab a sentence that out-salads
   my archetype.
2. Paste it **verbatim** into the issue's "This week's salad," with attribution:
   *— Company, form/call, date* (and a link).
3. Keep (or tweak) the existing Translation + What it's hiding.
4. Sanity check: is it a fair, short quote used for commentary? Yes → ship it.
